# JavaScript-Project
JavaScript course by Levi9 company, at the Faculty of Mathematics, University of Belgrade.


## Introduction
I have implemented a simple Maze Game using Vanilla JavaScript, HTML and CSS.

### Controls
| Key | Description|
| :-------------: | :-----------------: |
| `w` | move up |
| `a` | move left |
| `s` | move down |
| `d` | move right |
